Depot Patch Fix Builder
Đóng gói file fix → patches/{version}/ — launcher tự apply khi cài/update
Step 1 — Thư mục chứa file fix
Thư mục chứa đúng các file cần fix (DLL, exe…). Giữ nguyên cấu trúc thư mục giống bản gốc — ví dụ GameData\Plugins\fix.dll.
Step 2 — Repo root (nơi chứa catalog.json gốc)
Thư mục gốc của depot (nơi có catalog.json, manifests/, packs/). Tool sẽ tự tạo patches/{version}/ bên trong.
Step 3 — Cấu hình
Phải giống Game ID trong catalog.json gốc.
Tên version của patch này — dùng làm tên thư mục patches/v1.0/. Phải trùng với version trong main manifest.
Patch nhỏ nên để 64 MiB là vừa.
Step 4 — Cargo / Build
Key chỉ truyền qua env, không ghi vào file.
Output Log Idle
Tool này cần backend local Python.
Mở run-local-tool.bat hoặc chạy:
python 0xo_depot_uploader_server.py
Sau đó mở tool tại http://127.0.0.1:8776 thay vì mở file trực tiếp.
